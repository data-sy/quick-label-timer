## 계정/인프라

### 1. 구글 계정을 오드머핀 공용 계정으로 이관
- **배경:** Firebase 프로젝트(`quicklabeltimer`)와 오류 제보용 구글 설문이 개인 계정 소유로 되어 있음. 공용 계정(오드머핀)으로 이관해 개인 계정 의존을 없앤다.
- **작업:**
  - [ ] Firebase 콘솔 → 프로젝트 설정 → 사용자 및 권한에서 오드머핀 계정을 **소유자(Owner)** 로 추가 → 초대 수락 확인 후 기존 계정 제거
  - [ ] 오류 제보 구글 설문(`https://forms.gle/CobXgiRGjEFQZKgr8`)의 소유권을 오드머핀 계정으로 이전
    - ⚠️ **사본 생성 금지** — 사본을 만들면 forms.gle URL이 바뀌어 배포된 앱(`SettingsView.swift`의 문의 링크)이 깨진다. 반드시 Drive/Forms의 **소유권 이전**(공유 → 오드머핀 추가 → 소유자로 변경 → 상대 수락)으로 URL을 유지할 것. 응답 데이터도 함께 넘어가는지 확인.
- **관련:** 시크릿 스캐닝 얼럿 #1 (Google API 키 재생성) — 이관 후 오드머핀 계정에서 재생성하면 한 번에 정리 가능
